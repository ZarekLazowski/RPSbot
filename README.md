A bot that plays RPS against you

---

# Commands
| Command | Implemented | Function |
|---------|-------------|----------|
| /rps    | Partially   | Play RPS with the bot |
| /ratio  | No          | Display W/L ratio with user (can specify user) |
| /flip   | No          | Flip a coin, respond with "Heads" or "Tails" |
