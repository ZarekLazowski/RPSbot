# Commands
| Command | Implemented | Function |
|---------|-------------|----------|
| /rps    | Partially   | Play RPS with the bot |
| /stats  | Partially   | Display W/L ratio with user (can specify user) |
| /flip   | Yes         | Flip a coin, respond with "Heads" or "Tails" |
